## ✍ Short Description

This project is an API maked with mysql. In this project you can create, see, update and delete users and students. You can do login to, authenticating with user and password.

## 🧪 Technologies

This project was developed using the following technologies:

- NODEJS
- MYSQL
- JWT
- SEQUELIZE
